Listas con HTML


Listas ordenadas


Las listas ordenadas son las listas que muestran elementos en un orden

Ejemplo

  1. Elemento 1
    1. Sub Elemento 1
    2. Sub Elemento 2
    3. Sub Elemento 3
  2. Elemento 2
  3. Elemento 3

Listas desordenadas

  • Elemento 1
    • Sub elemento 1
    • Sub elemento 2
    • Sub elemento 3
    • Sub elemento 4
  • Elemento 2
  • Elemento 3
